The engine

What happens when you press Super+F2

A root-side systemd service fires. Not a script calling sudo. Not a udev hack. A proper polkit-backed service that writes directly to the kernel's CPU frequency files and AMD/NVIDIA power interfaces — in under a second.

The user service picks it up and swaps your Hyprland layout, kills blur and animations, and sends you a notification.

axiom-engine 
# Super+F2 pressed → mode = game

[engine-system]
CPU governor    → performance
AMD GPU         → high
steam gamescope → renice -5

[engine-user]
layout          → dwindle
blur            → off
animations      → off
rounding        → 4px

notify          → Game Mode — Performance unlocked
