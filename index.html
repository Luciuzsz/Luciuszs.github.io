v1.3 — what's new

Not just a rice.
A real system.

Every layer of AxiomOS ships actual tooling. Free rices look good in screenshots. AxiomOS changes how your computer actually runs.

axiom-motd

7 rotating ASCII art pieces — black hole, accretion disk, pulsar, solar system, wormhole, starfield, big-text. Different every session.

Quickshell bar

QML-native, replaces Waybar entirely. Mode chip animates on switch. Workspace pill-dots. Live CPU/RAM/temp. Reads theme reactively.

Orbital lockscreen

Two concentric ring decorations. 72pt clock. Blurred screenshot background. Live axiom mode label. Reads your state files.

Glass wlogout

Six logout buttons with per-action Tokyo Night accent glows. Blue=lock, purple=logout, teal=suspend, red=shutdown. CSS blur from compositor.

Starship prompt

Your axiom mode at every prompt. Git branch, status, command duration. Tokyo Night palette. Works in fish, bash, and zsh.

fastfetch layout

Axiom black hole logo in Tokyo Night alongside a clean, bordered system info panel. Color row at the bottom. Looks like a product, not defaults.

Axiom Engine — Full + Supporter only

Press a key.
Everything changes.

A root-side systemd service with a polkit rule. CPU governor, GPU power profile, process priorities, window layout, blur, and animations all flip in under a second. No sudo prompts, no hacks.

work

balanced CPU governor · dwindle layout · blur on · animations on

game

performance governor · high GPU profile · Steam/Wine renice to -5 · blur off · animations off

safe

powersave governor · master layout · auto-activates at 90°C or 20% battery

01

axiom-engine.service (root)

Sets CPU governor, GPU power profile. Renices steam and wine processes.

02

axiom-engine-user.service

Sends Hyprland IPC: swap layout, toggle blur, kill animations, fire notification.

03

polkit rule

axiom group members switch modes without a password. Ever.

04

Per-app auto-mode

Add steam=game to rules.conf. Steam opens → mode switches automatically.

05

Guards

Thermal guard: Safe mode at 90°C. Battery guard: exits Game at 20%.
